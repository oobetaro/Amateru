0.0.01a - 2007/12/25

  • 最初のリリース。非公開α版

1.0.0b - 2008/08/08

  • 一般に初公開。しばらくはβ版として開発を続ける。

1.5.0b - 2009/12/20

1.6.0b - 2011/07/24

  • VistaやWin7で起動に失敗するバグを修正。
  • 出生データに観測地が登録されていても、チャートに反映されなかったバグを修正。

2.0.1b - 2011/08/12

  • Java WebStartで配布することをやめ、専用インストーラを用意した。
  • サビアン等のフキダシ表示でフォントスタイルとサイズを変更可能にした。
  • サビアン辞書のフォントスタイルとサイズを変更可能にした。
  • レポートとアラビックのチャートで、画像ファイルによる占星術シンボルを廃し、占星術フォントを使った表示にした。
  • カラー設定(スキン)を数種類用意。
  • チャートのリング幅の初期値を広くした。
  • アラビックチャートに印刷機能をつけた。
  • 進行法の初期値を１度１年法から１日１年法にした。
  • 占星術辞典の辞書ファイルを最初から開くようにした。
  • ミチテルの祝日データを更新。
  • その他、五十箇所以上のバグ取りと仕様変更。
  • 全体的なリファクタリングによる最適化。

2.0.5 - 2020/12/18

  • なんと9年ぶりの更新。
  • 起動しなくなった問題を解決。
  • ミチテルとエニアグラムカレンダーの暦範囲を2040年まで増やした。
  • インストーラーが作るショートカットが起動しなくなっていたのを解決。
